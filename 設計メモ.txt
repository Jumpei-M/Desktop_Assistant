📋 デスクトップ常駐アプリ概要
🎯 目的
Windows 環境で常時稼働する、画面左端の「半月型のでっぱり」ウィジェットを作成

マウスカーソルを合わせると展開し、タスク一覧やメモ帳が表示される

今後も機能追加しやすい構成にする

🔷 表示内容（初期機能）
✅ タスク一覧
✅ 簡易メモ帳
（JSON ファイルにデータ保存）

🔷 開発環境・技術スタック
項目	選択
OS	Windows
開発言語	C#
フレームワーク	WPF (.NET 6 または 7)
データ保存	JSON ファイル
UIデザイン	Visual Studio 2022 Community（無料）
ロジック実装	VSCode

🔷 開発の進め方
1️⃣ Visual Studio
　UI（半月型ウィンドウ、タスクとメモを表示する領域）のデザイン・配置
2️⃣ VSCode
　イベントロジック、JSON読み書き、機能の実装
3️⃣ 運用
　Visual Studio でレイアウトを微調整しつつ、内部処理は軽快に VSCode で進める

🔷 拡張性のために
MVVMパターンを意識したコード構造にする

タスク・メモ以外のタブやパネルを追加しやすいようにする
